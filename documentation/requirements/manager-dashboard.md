# Manager Dashboard

## Overview
Web-based dashboard for property managers. This is the primary selling point — the screen that convinces managers to subscribe to Propmann.

## Key Screens

### Dashboard Home
The "wow" screen. At-a-glance overview of all managed properties.
- Property selector / switcher (for multi-property managers)
- Occupancy summary (occupied vs vacant units)
- Open maintenance requests count and urgency breakdown
- Upcoming rent due / collection status
- Recent activity feed (new requests, payments received, move-ins)
- Quick action buttons (post announcement, add resident, create request)

### Maintenance Requests
- List view with filtering (status, priority, property, date)
- Request detail view (description, photos, timeline, assigned contractor)
- Status workflow: New → In Progress → Completed → Closed
- Ability to assign to staff or contractors
- Communication thread with resident

### Resident Management
- Resident directory (searchable, filterable by property/unit)
- Resident profile (contact info, unit, lease dates, payment history)
- Onboard new resident (assign to property + unit, send invite)
- Offboard resident (end lease, vacate unit)

### Financial Overview
- Rent collection summary (paid, pending, overdue)
- Arrears tracking with resident details
- Monthly income/expense chart
- Payment history log

### Announcements
- Create announcement (select target: single property or all)
- Announcement history
- Read receipts / delivery status

### Amenity Management
- List of amenities per property
- Booking calendar view
- Add / edit / disable amenities
- View upcoming bookings

### Property Management
- List of managed properties
- Property detail (address, unit count, occupancy, photo)
- Add / edit property
- Unit listing per property (unit number, floor, status, current resident)
