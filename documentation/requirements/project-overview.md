# Propmann — Project Overview

## Product Vision
Propmann is a SaaS property management platform. Property managers subscribe to the platform to manage their properties and residents. The platform operator (Admin) runs the SaaS business, onboarding managers and handling billing.

## SaaS Model
- **Admin** operates the Propmann platform, onboards managers on subscription plans
- **Managers** each manage one or more properties from a web dashboard
- **Residents** interact through a mobile-first experience

## Three Interfaces
| Interface            | Primary User | Platform     | Priority |
|----------------------|-------------|--------------|----------|
| Management Dashboard | Managers    | Web          | Phase 1  |
| Resident Portal      | Residents   | Mobile-first | Phase 1  |
| Admin Console        | Admin       | Web          | Phase 2  |

## Demo Scope
This is a demo project. The primary goal is **visual functionality that conveys value to the client**. The demo should:
- Look polished and production-ready
- Show the two-sided value (manager manages, resident benefits)
- Demonstrate multi-property capability
- Prioritize breadth of features over depth (show many screens, not deep workflows)

## Multi-Property
The system is multi-property from the start. A single manager can manage multiple buildings/properties, each with their own set of residents, amenities, and maintenance history.
