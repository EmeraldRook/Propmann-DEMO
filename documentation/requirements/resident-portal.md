# Resident Portal

## Overview
Mobile-first interface for residents. Should feel like a modern consumer app — clean, fast, and intuitive. This is what managers show their residents as a selling point for the property.

## Key Screens

### Home
- Welcome / greeting with property name
- Quick action cards (pay rent, submit request, book amenity)
- Active maintenance request status (if any)
- Latest announcements (preview)
- Upcoming bookings

### Maintenance Requests
- Submit new request (category, description, photo upload, urgency)
- My requests list with status badges
- Request detail with timeline and communication thread
- Categories: Plumbing, Electrical, HVAC, Appliance, General, Other

### Payments
- Current balance / next payment due
- Pay now button (for demo: simulated payment flow)
- Payment history list
- Receipt view

### Announcements
- Feed of announcements from management
- Detail view with full content
- Mark as read

### Amenities
- Browse available amenities with photos
- Calendar view showing availability
- Book a time slot
- My bookings list with cancel option

### My Profile
- Personal info (name, contact, unit)
- Lease details (start date, end date, rent amount)
- Documents (lease agreement, house rules — view/download)
- Notification preferences

### Messages
- Conversation thread with management
- Send text message
- View message history
